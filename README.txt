This is a collection of simple Java programs to mess around with.

Current Programs:
    Gibberish: Creates a string of random gibberish of a certain length. (working)
    Prime: Inputs a number and determines if it is prime. (working)
    PronounceableGibberish: Creates a string of (hopefully) pronounceable english characters. (not working)
    RecursiveFactorial: Inputs a number and determines the factorial with a recursive function. (working)
